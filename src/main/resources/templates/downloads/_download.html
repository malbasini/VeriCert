File di supporto Vercert

Vercert è una piattaforma web per aziende che consente di emettere, gestire e verificare certificati digitali tramite QR code e identificativi univoci. Il sistema è progettato per garantire autenticità, tracciabilità e controllo dello stato del certificato nel tempo. Prima di emettere certificati o fatture è necessario compiere alcune operazioni fondamentali . La prima volta che si entra nel portale occorre creare l'utente Admin e il tenant che gestirà insieme ai suoi collaboratori. Per fare questo occorre fare il signup, tale operazione oltre a registrare l'utente Admin del tenant abilita alcune voci di menù tra cui Admin. Per usare il software occorre compiere alcune operazioni preliminari entrando nella sezione Admin e compilando le informazioni richieste nelle Impostazioni, compilare le varie tab, logo, firma per i certificati, i dati aziendali del fornitore per le fatture. In questa pagina è possibile scaricare un file zip contenente i template HTML e i file Json. Per ulteriori informazioni fare riferimento alla documentazione di vercert.
